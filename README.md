**Input**:</br>
  <span>  **p**, principal amount</br></span>
  <span>  **t**, time period in years</br></span>
  <span>  **r**, annual rate of interest</br></span>
   
**Output**:</br>
   </span>  __simple interest__ = p\*t\*r</span>
